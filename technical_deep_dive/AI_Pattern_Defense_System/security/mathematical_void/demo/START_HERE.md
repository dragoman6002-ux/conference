## ğŸš¨ Troubleshooting

### "Address already in use"

**Solution:**
```bash
# Kill existing server
pkill -f demo_server.py

# Or use different port
python demo_server.py 9000
```

### "Connection refused"

**Solution:**
1. Verify server is running: `curl http://localhost:8000`
2. Check no firewall blocking port 8000
3. Try localhost vs 127.0.0.1

### "creator_token.txt not found"

**Solution:**
1. Make sure server started successfully
2. Check current directory
3. Token created on server startup

### All requests getting decoy

**Solution:**
1. Check token is correct: `cat creator_token.txt`
2. Verify header format: `Authorization: Bearer <token>`
3. No extra spaces in header
