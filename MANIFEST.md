## üìû Support & Questions

### If something doesn't work:

1. **Run verification first:** `python verify_connections.py`
2. **Check Python version:** `python --version` (need 3.6+)
3. **Test tools individually:** `python <tool_name>.py`
4. **Check tkinter:** `python -c "import tkinter; print('OK')"`

### Common Issues:

**"tkinter not found":**
- Windows: Reinstall Python with tcl/tk checked
- Linux: `sudo apt-get install python3-tk`
- Mac: `brew install python-tk`

**"Tool won't execute":**
- Check file permissions
- Try: `python infinite_thought.py` directly to see errors

**"No output in GUI":**
- Look at terminal where GUI was launched
- Errors often appear there
